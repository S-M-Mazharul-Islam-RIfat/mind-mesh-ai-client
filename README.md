# Project Local Setup Guideline (Frontend)


## Step 1. Clone the Repository

```bash
git clone https://github.com/S-M-Mazharul-Islam-RIfat/mind-mesh-ai-client.git
cd mind-mesh-ai
```
---

## Step 2. Install dependencies
```bash
npm install
```

---
## Step 3. Run the frontend
npm run start:dev <br>
npm run start:worker
